# subscribe-quote
订阅行情数据

通过程序订阅股票实时数据，例如如下命令可订阅 SH600000 股票实时数据
```
./tstClient SH600000
```
数据信息包括当前K线的开高低收、当前盘口的5档数据等等

例如，如下命令可订阅标普500指数
```
./tstClient IXSPX
```

## Linux 下编译方法
在 ubuntu 18.04 验证通过

* 编译

安装依赖包
```
sudo apt-get install libmosquitto-dev zlib1g-dev
```

编译
```
cd sub-quote-cpp
cmake .
make
```

* 运行
```
./tstClient
```

* 注意
默认体验密码和服务器地址都已经在程序中固定，有需求可以联系我们获取独立的用户名密码
